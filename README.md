# ü©∏ Blood Bank Management System (Core Java)

This is a console-based Blood Bank Management System developed using Core Java.
The project helps in managing donors, patients, blood inventory, and blood requests
using file handling for data storage.

## Project Objective

To provide a simple and efficient way to manage blood bank operations using
Core Java without using any database.

## Modules in the Project

### Admin Module
- Admin Login
- Donor Management (Add, View, Update, Delete)
- Patient Management (Add, View, Update, Delete)
- Blood Inventory Management
- Logout

### User Module
- User Login
- View Available Blood Stock
- Request Blood
- Logout

### Data Management Module
- Store donor details using text files
- Store patient details using text files
- Store user and admin credentials
- Persistent data using Core Java File Handling

## Technologies Used
- Core Java
- OOP Concepts (Class, Object, Constructor)
- File Handling (Text Files)
- Scanner Class
- Eclipse IDE

All data is persistent and remains saved even after program restart.
